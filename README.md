# UIKit Vue Components

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for demo page
```
npm run build
```

### Compiles and minifies for library plugin
```
npm run build:lib
```

### Lints and fixes files
```
npm run lint
npm run lint:css
```

### Run your unit tests
```
npm run test:unit
```
